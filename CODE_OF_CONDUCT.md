# Code of Conduct

## Our Pledge

We want this project to be useful, respectful, and welcoming for contributors with different levels of experience.

Participants are expected to:

- Use welcoming and inclusive language.
- Respect different viewpoints and experiences.
- Give and receive technical feedback constructively.
- Focus criticism on ideas, code, and documentation rather than people.
- Avoid harassment, insults, discrimination, or disruptive behavior.

## Scope

This Code of Conduct applies in project spaces, including issues, pull requests, discussions, and any other public collaboration connected to this repository.

## Reporting

If you see behavior that violates this Code of Conduct, report it to the repository maintainer through GitHub. Include relevant links, screenshots, or context when possible.

Maintainers may remove comments, close issues, block users, or take other reasonable action to keep the project healthy.

## Enforcement

Maintainers will review reports in good faith and respond in a way that is proportionate to the situation. Repeated or severe violations may result in temporary or permanent removal from project spaces.
