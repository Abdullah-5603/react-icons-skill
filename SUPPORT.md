# Support

This is an unofficial community Agent Skill.

For help:

- Use GitHub issues for bugs in the skill instructions, references, evals, or install documentation.
- Use feature requests for additional examples, framework guidance, accessibility patterns, or eval cases.
- Check `skills/react-icons/references/` before opening a question about icon prefixes, accessibility, Next.js, or SVG migration.

This project cannot provide official support for the React Icons package itself. For package bugs, icon availability, or upstream behavior, use the official React Icons project resources.
